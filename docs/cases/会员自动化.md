# 会员自动化

## 项目背景

会员自动化项目由高高提出，启动背景是门店收银流程混乱、操作方式不统一，同时原有处理步骤复杂。一线在开卡、升级、退卡、优惠等场景中需要做大量人工操作，不同同事的处理方式也不一致。

在领健系统中，会员、卡券权益、收费彼此独立，缺少自动对应关系。对一线来说，这意味着收银流程繁琐、容易出错；对财务来说，这意味着月底核对收入、优惠和会员权益时需要大量人工比对，管理成本高。

## 项目目标

- 简化一线操作时长，提升效率。
- 规范会员优惠流程，避免不必要的折扣方式。
- 建立会员、卡券、收费、账单之间的自动对应关系。
- 降低财务核对和审查成本。

## 我的角色

我在这个项目中同时承担产品设计、产品经理和项目推进者的角色。

我主要负责：

- 推动开发进度。
- 上线后会员优惠券的维护与梳理。
- 上线后的运营跟进与问题修改。
