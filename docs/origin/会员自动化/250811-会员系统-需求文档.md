# 250811-会员系统-需求文档

# 用户故事

a.作为运营，我希望【升级/续/退会员卡时，系统自动帮我处理账单和开卡】，以便【减少升级/续/退会员卡时的操作时间】，体现在【升级/续/退会员卡时的操作时间】

b.作为运营，我希望【升级会员卡时，系统自动帮我计算有效期】，以便【减少遗漏修改会员卡有效期】，体现在【升级会员卡的有效期准确率】

c.作为运营，我希望【在调整卡的价格时，可以在合理范围内调整优惠力度】，以便【可灵活操作，无需每次都走审批】，体现在【收费的时间】

d.作为店长，我希望【系统可以设置运营的优惠额度并跟踪使用记录】，以便【规范门店优惠流程，减少滞后审批操作，避免绕过审批先执行优惠】，体现在【未经审批的优惠数量】

# 产品目标

*   减少手工操作，提升前台效率
    
*   降低培训与操作成本
    
*   保证卡种、账单、权益的自动一致性（哪些人可以用哪些卡券和权益）
    
*   规范价格修改流程，引入额度分配机制，提升合规性与灵活性

*   **临时：**
    
    *   入口：左侧菜单账单会员下二级菜单“会员管理”
        
    *   列表：
        
        *   日期默认选中当天，可切换日期，必选
            
        *   预约医生默认不选中。单选，不选中医生时，列表展示当前登录诊所和对应日期的所有就诊患者；选中医生时展示当前登录诊所和对应日期的该医生的就诊患者。
            
        *   列表单页超过10条时分页展示。
            
        *   点击右侧“查看会员详情”，进入会员页面。

#### 升/续卡

*   点击"升/续卡"按钮出现弹窗，弹窗逻辑与**【4.3.2.1开通新卡】**一致[《250811-会员系统-需求文档》](https://alidocs.dingtalk.com/i/nodes/EpGBa2Lm8azQjxMnC0k7DeRAWgN7R35y?utm_scene=person_space&iframeQuery=anchorId%3Duu_me16cfbzb87nu0uuy5s)

*   **升/续卡规则：**
    
    *   一年卡可以升级到两年卡，一年卡/两年卡可以升级到五年卡（除恒橙卡外），恒橙卡可以升级到五年卡；
        
    *   原卡有效期大于三个月且卡内还有项目，可以升级；
        
    *   **若判断为升级**
        
        *   选择优惠券和手动优惠
            
        *   根据优惠金额生成账单数据（支付金额为升级补差的金额）
            
        *   调用领健接口收费
            
        *   **同门店升级**
            
            *   使用当前诊所的领健账号获取token
                
            *   调用领健接口退原会员卡账单的项目
                
            *   调用领健接口收欠费
                
        
        *   **不同门店升级**
            
            *   使用原账单诊所的领健账号获取token
                
            *   调用领健接口退原会员卡账单的项目
                
            *   调用领健接口收欠费（支付金额为退项目的金额）
                
        *   **立即开卡**
            
            *   调用领健接口升级会员卡（修改有效期）
                
            *   如果原卡是涂氟卡/橙宝卡/小橙宝卡/大橙宝卡/水晶卡
                
                *   会员卡券卡内项目未使用过，调用领健接口注销卡券
                    
                *   会员卡券卡内项目已使用过，调用领健接口修改卡券有效期
                    
        
        *   **查询到多个原账单**
            
            *   则右下展示“升级方式购卡，查询到多个现有会员卡账单，请选择退费账单”![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/4EZlweZzDaarLqxA/img/25210520-1d9b-4e76-b747-b244516325ae.png)![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/4EZlweZzDaarLqxA/img/49f6f4c1-915c-450b-8a47-c1344dc66a9d.png)
