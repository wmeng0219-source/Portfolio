# 收银补充手册（ODOS试行版）

# 关于ODOS系统规范使用的通知 

各位同事： ODOS系统上线运行已有一段时间，为进一步规范系统操作流程、明确使用要求，保障会员业务有序开展，现将相关使用规则强调如下：

###  会员业务全流程线上化操作 

所有会员功能相关操作，均需在ODOS系统内完成，涵盖但不限于会员卡开卡收费、优惠券发放、会员等级升级、会员卡退费等业务场景。

### 活动信息的核实与维护 

所有常规活动及市场活动，若未在ODOS系统中查询到相关信息，可向IT部门提交核实申请。经IT部门确认活动真实存在且处于有效期内的，将由IT部门统一在ODOS系统中完成信息维护。

###  会员收费额度管理及违规处理 

（1）**额度分配原则：**ODOS系统会员收费额度，依据运营端会员卡标的配比进行分配。额度使用完毕后不再追加，严禁擅自向IT部门申请增加额度。

（2）**特殊额度申请流程：**针对政府关系维护等极特殊情况，可凭大卫批复截图，向IT部门申请临时增加收费额度，除此之外无其他特殊审批渠道。 

（3）**违规操作界定：**严禁在领健系统中操作会员卡收费业务，亦不得在领健系统发放ODOS卡券。上述行为一经发现，均视为收银违规。 

（4）**违规处罚标准** 

① 违规发生后，将按照违规减免金额，按照运营50%、主管及店长35%（均摊）、区域管理层15%（区域管理层内均摊）的比例，在责任人当月工资中扣除对应款项；且次月对该笔违规金额进行双倍额度扣除。 **② 于领健系统中违规操作的会员卡收费业务，不计入任何绩效核算范围。**

*   点击共优惠⑨可以查看优惠明细
    
*   点击确认开通新卡⑩

    ### 升级会员卡
    
    在会员详情页 - 升级会员卡。
    
    **升级条件判断：**
    
    *   现有卡为短期卡（一年卡/两年卡）。**注：小橙宝卡不可升级水晶卡**
        
    *   恒橙卡可以升级金橙卡（特殊）
        
    *   购买新卡为长期卡（比原卡有效期长的卡）
        
    *   现有卡有效期逾期在6个月内，卡内有剩余项目
