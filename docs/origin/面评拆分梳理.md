# 流程对比

## 当前现状

1.  重复筛选是否需要矫正的患者：快筛、面评会、儿牙面评后不确定无法下
    
2.  提前过面评会有遗漏：面评会过未来一周的所有患者，临时患者无法被筛选到。
    
3.  面评会基于历史的就诊数据，可能不准确。
    
4.  儿牙医生下面评结论需正畸医生协助，当前面评表不支持多人分步完成面评的场景。

## 改版后流程

1.  调整面评推荐逻辑，取消快筛，减少重复操作
    
2.  根据系统推荐，儿牙医生直接面评，面评后会诊直接下结论进行下一步操作（或当天转正畸医生）

### 外院矫正的后续流程？

在儿牙部分正畸筛查时勾选外院矫正，当次儿牙医生告知我们定期检查也会关注小朋友的咬合，系统推送1年后复查咬合 

### 快筛取消后，快筛无结论的患者应该按照什么逻辑推荐面评？

1.  无结论视为未曾面评 儿牙部分推荐咬合筛查，儿牙医生完善资料后面评会正畸医生录入评估结果确认后续计划。 
    
2.  开始的时候量会很大 面评推荐优先级我们需要定一下 依次：恒牙列、剩下四个乳牙、2-2萌出完毕、上1萌出完毕、其余替牙期 
    

### 快筛取消后的面评的理想推荐逻辑是什么？

1.  有之前面评结论和下次跟进计划的（面评表）按照结论和跟进时间推荐；无结论的视为未曾面评。
    
2.  4岁建议第一次乳牙期咬合筛查，后续根据结论跟进日期推荐
    
3.  第一个1或者6萌出后第一次替牙期咬合筛查，后续根据结论跟进日期推荐，最少两年一次
    
4.  剩余四颗乳牙的时候进行恒牙期咬合筛查，后续根据结论跟进日期推荐，最少两年一次 
    
5.  数量比较大的时候优先级：恒牙列高于剩余四颗乳牙高于2-2萌出完毕，高于两个1，高于无1，高于乳牙期 
    
6.  早期矫正结束后在审批流程中加入下一阶段评估的时间，到时间后做这个动作。
