# 站点说明文档

本目录用于说明当前作品集网站本身的结构与内容组织方式。

## 站点定位

这是一个**用于求职展示的个人网站**。

- 主要目标：服务求职场景，帮助招聘方、面试官或潜在合作方快速理解个人背景、项目经验与能力方向。
- 内容取舍：优先保留与求职相关的信息，包括个人介绍、经历、项目案例、方法论与联系方式。
- 表达原则：以清晰、专业、可信为优先，避免偏离求职目标的冗余内容或无关功能。

## 文件说明

- `structure.md`：当前网站项目目录结构与职责说明。
- `content-map.md`：背景资料、项目案例与页面区块的映射关系。

## 使用建议

- 需要了解网站目录时，先看 `structure.md`。
- 需要补充页面内容时，先看 `content-map.md`，再回到 `docs/profile/` 和 `docs/project/` 查找对应内容。
- 需要查看设计与计划过程文档时，前往 `docs/superpowers/`。
