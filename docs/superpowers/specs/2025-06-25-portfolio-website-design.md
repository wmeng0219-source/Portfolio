# Portfolio Website — "数字化叙事者" 设计规格

## 概述

为 Meng Wen 构建一个求职导向的个人作品集网站，定位为**产品设计复合型人才**（UI/UX + PM），以滚动叙事驱动的单页形式展示从 UI 设计师到产品经理/ITBP 的职业转型故事。

**目标受众：** 潜在雇主、HR、招聘方  
**核心定位：** 产品设计师 · 数字化实践者  
**语言：** 中英双语可切换  

---

## 技术架构

| 维度 | 方案 |
|------|------|
| 框架 | Vite + React |
| 动画 | GSAP + ScrollTrigger |
| 样式 | Vanilla CSS（CSS Variables + CSS Modules），深色主题 |
| 字体 | Inter（英文）+ Noto Sans SC（中文），通过 Google Fonts 加载 |
| i18n | React Context + JSON 文案文件（`locales/zh.json`、`locales/en.json`） |
| 响应式 | 桌面优先（Desktop-first），适配平板（768px）和手机（375px） |
| 部署 | 静态构建，可部署到 Vercel / Netlify |

## Section 1: Hero 首屏

### 布局

全屏高度（100vh），左右分栏（6:4 比例）。

- **左侧（60%）：** 文字内容区
  - 问候语："你好，我是"（小字，辅助色）
  - 名字："Meng Wen"（超大字号，渐变色文字，加粗）
  - 定位语："产品设计师 · 数字化实践者"（中等字号，辅助色）
  - 个人宣言：引用核心理念（斜体/引号样式）
  - CTA 按钮组：「查看作品」（渐变填充主按钮）+「联系我」（描边次按钮）

- **右侧（40%）：** 视觉元素区
  - 个人照片（圆角裁剪 + 微光渐变边框）
  - 如暂无照片，使用抽象浮动几何图形动画（圆环、光点粒子）

- **底部居中：** 滚动提示箭头（循环浮动动画）

### 入场动画（GSAP Timeline）

```
t=0.0s  名字从下方淡入滑上
t=0.2s  副标题淡入
t=0.4s  个人宣言淡入
t=0.6s  CTA 按钮从下方弹入
t=0.3s  右侧视觉元素缩放淡入
t=1.0s  滚动提示开始循环动画
```

### 移动端

- 左右分栏 → 上下堆叠（视觉元素在上，文字在下）
- 字号适当缩小（已通过 clamp 处理）
- CTA 按钮全宽

- 左右分栏 → 上下堆叠

## 语言切换（i18n）

### 实现方式

- React Context（`LanguageContext`）管理当前语言状态
- 提供 `useLanguage()` Hook 获取当前语言和切换函数
- 所有文案通过 `t('key')` 函数获取
- 文案存储在 `locales/zh.json` 和 `locales/en.json`
- 切换时无页面刷新，组件实时更新
- 默认语言：中文
- 用户选择保存到 localStorage

## 性能优化

- 图片使用 WebP 格式 + 懒加载（Intersection Observer）
- GSAP 动画仅在元素进入视口时触发
- 字体使用 `font-display: swap` 避免 FOIT
- CSS 使用 CSS Modules 避免全局冲突和多余样式
- Vite 自动代码分割

---

## 可访问性

- 语义化 HTML（header、main、section、footer）
- 所有交互元素有 focus 样式
- 图片有 alt 文本
- 颜色对比度符合 WCAG AA 标准
- `prefers-reduced-motion` 媒体查询：禁用动画时降级为简单淡入
